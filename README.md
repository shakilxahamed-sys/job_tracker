# 💼 Job Application Tracker

A real-time web application built with Python and Streamlit 
to help job seekers track their applications efficiently.

## 🚀 Live Demo
[Click here to view the app](#) 
← (we will add the link after deployment!)

## 📌 Problem It Solves
Most job seekers apply to multiple companies and lose 
track of their applications. This app solves that by 
giving a single dashboard to manage everything!

## ✨ Features
- ➕ Add new job applications
- 📊 Visual dashboard with charts
- ✏️ Update application status
- 🗑️ Delete old applications
- 🔍 Filter by status

## 🛠️ Tech Stack
| Tool | Purpose |
|------|---------|
| Python | Backend logic |
| Streamlit | Web interface |
| Pandas | Data management |
| Plotly | Charts & visualization |
| CSV | Data storage |

## 🔧 How to Run Locally

1. Clone the repository
git clone https://github.com/YourUsername/tracker_job.git

2. Install dependencies
pip install -r requirements.txt

3. Run the app
python -m streamlit run app2.py

4. Open browser at
http://localhost:8501
